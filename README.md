# GPGWizard

GPGWizard is a tool for generating GPG keys with the following functionalities:

- Lists GPG keys
- Deletes GPG keys
- Adds GPG keys to Git
- Creates GPG keys

## How to Use

To use GPGWizard, follow these steps:

   ```bash
   git clone git@github.com:blackpanther26/GPGWizard.git
   cd GPGWizard
   ./gpg.sh
